設定名稱：git config --global user.name "名稱"
設定信箱：git config --global user.email "信箱"
下載專案：git clone <網址>      
檢查狀態：git status	       
加入修改	：git add .	     
確認存檔：git commit -m "訊息"  
上傳雲端：git push	        

下載更新	：git pull
1. Fast-Forward：別人版本未修改到自己的部分 -> 不用解
2. CONFLICT：別人的版本和自己內容不符 -> 開啟檔案，手動解 conflict	       
創建分支：git branch feature/功能名
進入分支：git switch feature/功能名
編輯分支...
回到 main：git switch main
下載更新	：git pull
進入分支：git switch feature/功能名
合併最新：git merge main
若有衝突自己解
上傳分支：git push -u origin feature/功能名
去 GitHub 網頁點 "Create Pull Request"
貼連結給同事：「幫我 Review 一下功能。」
同事看完並按下了 Merge 按鈕

查看註解：git log
查找細節：git show --stat <Commit-ID 前6~7碼即可>
